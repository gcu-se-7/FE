## 🔧 기술 스택

- ⚛️ React (CRA 기반)
- 🎨 Tailwind CSS
- 🌐 React Router



## 🚀 주요 기능

- 🥬 식재료 입력 기반 레시피 추천
- 🍚 한식 소개 페이지 (인트로)
- 🌈 Tailwind를 이용한 반응형 스타일
- 🔗 상단 네비게이션 메뉴로 페이지 이동
